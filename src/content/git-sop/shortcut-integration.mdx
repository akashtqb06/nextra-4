# Shortcut Integration

## Story-Driven Development

Our Git workflow is tightly integrated with Shortcut for comprehensive project tracking and traceability.

```
Shortcut Integration Flow

┌─────────────────────────────────────────────────────────────────────────────┐
│                           SHORTCUT WORKFLOW                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  Story Created → Story Started → Branch Created → Development → PR → Done   │
│                                                                             │
│  ┌───────────┐  ┌───────────┐  ┌───────────┐  ┌───────────┐  ┌───────────┐ │
│  │  Shortcut │  │  Update   │  │    Git    │  │   Code    │  │ Shortcut  │ │
│  │   Story   │  │  Status   │  │  Branch   │  │   Review  │  │  Story    │ │
│  │  Created  │─▶│"Started"  │─▶│  Created  │─▶│    &      │─▶│ "Complete"│ │
│  │           │  │           │  │           │  │   Merge   │  │           │ │
│  └───────────┘  └───────────┘  └───────────┘  └───────────┘  └───────────┘ │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Shortcut Story States

1. **Unstarted** - Story created, awaiting assignment
2. **Started** - Developer assigned, work begins
3. **In Development** - Active development, branch created
4. **Ready for Review** - PR created, awaiting review
5. **In Review** - Code review in progress
6. **Ready for Testing** - Merged to master, testing required
7. **Done** - Story completed and deployed

## Branch-to-Story Mapping

```bash
# Always include Shortcut story ID in branch name
git checkout -b feature/sc-12345-user-authentication
```

## Commit Message Integration

Use these keywords in commit messages for automatic story updates:

- `Closes`/`Fixes`: Marks story as complete
- `References`: Links commit to story
- `Updates`: Adds comment to story

## Pull Request Integration

Use the PR template to link stories and document changes.
